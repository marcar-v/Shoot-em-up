# Shooter 2D — Unity

## Descripción
Shooter 2D desarrollado en Unity centrado en sistemas de combate, optimización y gestión de entidades. El proyecto pone el foco en el uso de *object pooling* y en la escalabilidad del sistema de disparo.

## Características principales
- Sistema de disparo del jugador.
- Ataque especial con entrada dedicada.
- Pool de proyectiles mediante *Object Pooling*.
- Enemigos con disparo propio y comportamiento progresivo.
- Sistema de puntuación, vidas y *high score*.
- Fondo con *scroll* continuo.
- Menú principal, menú de pausa y pantalla de *Game Over*.
- Audio integrado (música y FX).

## Tecnologías
- Unity
- C#
- Object Pooling
- UI Canvas
- Audio Source

## Estado del proyecto
Proyecto finalizado y optimizado para su presentación en portfolio.
